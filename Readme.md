Никита Зайка
========
Что это будет?
--------------
Веб-сайт "FansPen"- иновационная платформа для создания и просмотра фанфиков.

Сайт будет отображать рейтинг фанфинков. 


Средства разработки
-------------------
Сервисы:
- Microsoft Azure;
- Google API.

Языки программирования:
- C#
- JavaScript.

Библиотеки:
- Hibernate; 
- jQuery;
- ASP.Net core 2
- Bootsrap

Базы данных:
- MS SQL server
